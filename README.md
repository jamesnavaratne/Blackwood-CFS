# Blackwood CFS v2.0 Stable Baseline

Stable baseline for the combined **Blackwood CFS Inventory + Directions Book** app, before the dedicated Directions Book fidelity-check phase.

## Included modules

- Blackwood Rescue inventory
- Blackwood 34P inventory
- CAFS 24 inventory
- Directions Book
- UBD Maps inside Directions Book
- Global Search across appliances
- Item Location Quiz

## Current stable behaviour

- Dark-only interface with Rescue Blue as the default theme.
- Theme palette: Rescue Blue, Safe Green, CFS Yellow, Brick Red.
- System home with appliance entry points.
- Hamburger menu sections: Appliances, Search, Directions, Theme Palette, Training.
- Global Search across all appliances.
- Item Location Quiz from the Training section.
- CAFS 24 P4 is included and ordered between P3 and D-Tube.
- Directions Book search bar stays pinned while scrolling.
- Directions Book street modal includes UBD, favourite, and close controls.
- UBD split-panel view keeps the street directions visible above the map list/viewer.
- UBD map images live in a replaceable folder structure.

## Inventory source of truth

The active workbook is:

```text
Blackwood_CFS_Master_Inventory.xlsx
```

Inventory data, appliance location names, and tab ordering are controlled from this workbook.

The `Appliance_Config` sheet controls locker/location tab order for each appliance. Home, Starred, and Out of Service are generated by the app and should not be added to `Appliance_Config`.

## Directions Book source of truth

The current Word source files are stored in:

```text
directions/source/
```

Included source documents:

```text
BLACKWOOD CORO EDEN HILLS ETC STREETS 122016.docx
CORO STREETS 122016.doc
CORO EAST & CHERRY STREETS 122016.doc
BLACKWOOD PARK STREETS.doc
```

The live Directions Book content is currently rendered from `directions/index.html`. For future updates, edit the relevant Word source document, then sync only the changed Directions Book content into the app.

Exact Word bold-run preservation is intentionally deferred to the dedicated Directions fidelity phase.

## Update workflow

### Inventory changes

1. Edit `Blackwood_CFS_Master_Inventory.xlsx`.
2. Regenerate the app data from the workbook.
3. Confirm each appliance opens correctly.
4. Confirm CAFS 24 P4 remains between P3 and D-Tube.
5. Deploy the regenerated package.

### Directions changes

1. Edit the relevant Word document in `directions/source/`.
2. Sync only the changed Directions Book content into the app.
3. Preserve existing UI and behaviour unless a separate UI change is requested.
4. Run the Directions fidelity checklist before calling Directions data final.

## Archive

The `Archive/` folder is for legacy reference files only. Do not use archived files as the active inventory source.
