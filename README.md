# Blackwood CFS v2.0 RC1

Multi-appliance inventory system for Blackwood Country Fire Service appliances.

## Appliances

- Blackwood Rescue
- Blackwood 34P
- CAFS 24

## Key features

- Multi-appliance inventory switching
- System home page
- Appliance home pages
- Locker/location tabs
- Starred items
- Out of Service view
- Item detail panels
- Locker and item photo support
- Global Search across appliances
- Training/quiz mode
- Offline-friendly static app structure
- Dark-only interface

## Inventory source of truth

The active workbook is:

`Blackwood_CFS_Master_Inventory.xlsx`

Inventory data, appliance location names, and tab ordering are controlled from this workbook.

The `Appliance_Config` sheet controls locker/location tab order for each appliance. Home, Starred, and Out of Service are generated by the app and should not be added to `Appliance_Config`.

## Photo structure

```text
photos/
├── rescue/
│   ├── lockers/
│   └── items/
├── 34p/
│   ├── lockers/
│   └── items/
└── cafs24/
    ├── lockers/
    └── items/
```

Displayed photo filenames are intended to show the exact path where photos should be placed.

## Archive

The `Archive/` folder is for legacy reference files only. Do not use archived files as the active inventory source.
