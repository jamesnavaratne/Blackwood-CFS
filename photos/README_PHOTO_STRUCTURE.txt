Blackwood CFS photo folder structure

Locker/location photos:
photos/rescue/lockers/
photos/34p/lockers/
photos/cafs24/lockers/

Item photos:
photos/rescue/items/
photos/34p/items/
photos/cafs24/items/

Use forward slashes in Excel/app paths, for example:
File: photos/34p/lockers/34p_d2.jpg

Keep filenames lowercase where possible.
Use .jpg unless you deliberately update the Excel/app path to .png.
