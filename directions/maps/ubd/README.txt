Blackwood CFS Directions Book - UBD Maps

To replace an existing map: replace the JPG with a file of the same name.
To add or remove a map: add/remove the JPG, then edit maps.js so the title and file name match.
The list shown in the Directions Book follows the order in maps.js.

Example entry:
  {"title": "UBD 169", "file": "UBD 169.jpg"}
