Blackwood CFS Directions Book source documents

These Word documents are the source-of-truth set used for future Directions Book content/fidelity checks.
The live app currently renders generated Directions Book content from directions/index.html.

Current source files:
- BLACKWOOD CORO EDEN HILLS ETC STREETS 122016.docx
- CORO STREETS 122016.doc
- CORO EAST & CHERRY STREETS 122016.doc
- BLACKWOOD PARK STREETS.doc

Suggested update workflow:
1. Edit the relevant Word document.
2. Keep the edited file in this folder using the same filename where possible.
3. Ask ChatGPT to sync only the changed Directions Book content into the app.
4. Keep UI/behaviour unchanged unless a separate UI change is requested.

Notes:
- Exact Word bold-run preservation is intentionally deferred to the dedicated Directions fidelity phase.
- Do not modernise wording into GPS style unless explicitly requested.
- Preserve old landmarks, response codes, map references, number ranges, dirt-road/fire-track notes, blocked ends, small-lane/fork notes, and original direction cadence.
