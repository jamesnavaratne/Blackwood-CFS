This version adds automatic item-photo fallback:
- If extraPhoto is set in the data, the app uses that filename.
- If extraPhoto is blank, the app tries photos/items/<sanitised_item_name>.jpg.

Example:
RCR CREW COMPOSITION x 4 (roof)
→ photos/items/rcr_crew_composition_x_4_roof.jpg
