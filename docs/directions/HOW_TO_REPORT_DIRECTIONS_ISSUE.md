# How to Report a Directions Book Issue

Each Directions Book entry now has a small `Report issue` pill at the bottom of the entry card.

## How to use it

1. Open the Directions Book.
2. Search for or browse to the street.
3. Tap the street entry.
4. Scroll to the bottom of the entry card.
5. Tap `Report issue`.
6. Choose the issue type:
   - Wrong directions
   - Missing detail
   - Map/code issue
   - Other
7. Add free text explaining what is wrong or what should be updated.
8. Tap `Send Email`.

The email draft includes:

```text
Street
Suburb/area
Book/source
Source file
Entry ID
Map/code
Current Directions Book wording
Your notes
```

## Purpose

This is designed for suggested updates to the source Directions documents, not for changing the app data directly.

The Directions Book remains source-document driven.
